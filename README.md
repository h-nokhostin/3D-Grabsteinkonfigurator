# 3D-Grabstein-Konfigurator

Ein einfacher 3D-Konfigurator für Einzelgräber auf Basis von Three.js. Die Startseite bietet eine Heldensektion mit direkter Aktionsleiste und einem Live-3D-Viewer zum Anpassen von Form, Material, Inschrift und Beleuchtung.

## Live-Link

Die aktuelle Demo ist über GitHub Pages verfügbar:

- **Live-Demo:** https://dein-github-benutzername.github.io/3D-Grabsteinkonfigurator/

## Lokale Nutzung

1. Repository klonen oder herunterladen.
2. Im Projektverzeichnis einen lokalen Server starten, z. B. mit Python:
   ```bash
   python3 -m http.server 8000
   ```
3. Im Browser `http://localhost:8000` öffnen und den Konfigurator testen.
